Enfoque Determinístico

El sistema tradicional se basa en la Iteración Lineal. Es la forma más eficiente de calcular Fibonacci para valores pequeños y medianos, garantizando un margen de error de cero. 

function fibonacci(n) {
  let a = 0, b = 1;
  for (let i = 0; i < n; i++) {
    [a, b] = [b, a + b];
  }
  return a;
}
  • Complejidad temporal O(n)
  • Consumo de memoria constante
